Rapide Node JS :

Serveur local basique utilisant la technologie Node JS & le Framework Express JS

Version actuelle stable

Version actuelle 1.0.0

A propos du projet

RapideNodeJS fonctionne comme une surcouche de Express JS. Le but du projet est de permettre au développeur débutant ou confirmé d'avoir un serveur Node JS basique facile à modifier et utiliser. Pour cela l'entièreté du projet est disponible sur Github avec la documentation. Les premières versions du projet seront créées seulement à but d'apprentissage et de test de la technologie.

Que dois-je faire ?

Si vous n'avez jamais utiliser NodeJS mais que vous connaissez JavaScript vous n'aurez pas trop de mal à vous y retrouvez.

Explication rapide

Le projet utilise l’architecture MVC. Vous avez ainsi le contrôleur qui envoie un ' Object ' à la vue que vous pouvez par la suite utiliser pour afficher ou traiter des informations.
Si vous avez plus de questions (comme par exemple comment l'installer) réfèrerez-vous à la documentation.

Qu’est-il possible de faire avec RapideNodeJS ?

  • Créer un site en utilisant le model MVC.
  • Créer rapidement une ou plusieurs pages pour son site.
  • Choisir l'url que l'on veut pour sa page ou même en mettre plusieurs.

! Il n'est pas recommandé de l'utiliser comme serveur de production mais seulement de test !

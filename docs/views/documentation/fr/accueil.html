Installations

Pour installer les dépendance importante du projet il vous faudra avoir NODE JS installer sur votre ordinateur. Si vous ne l'avez pas vous pouvez le télécharger directement sur le site officiel de Node JS.

Si vous avez déja Node JS installer sur votre système il vous suffit de lancer cette commande dans un cmd

CMD

npm i

Une fois cela fait ouvrir votre navigateur et aller a l'addresse http://localhost:4444
